## How to Use

### For Social Media
1. Choose the appropriate platform file (Facebook, Instagram, Threads)
2. Select an ad copy option that fits your campaign
3. Customize with your actual URL
4. Add your images/videos
5. Set up your ad campaign

### For Blog Posts
1. Choose the blog platform file
2. Copy the content
3. Replace `[your-url]` with your actual URL
4. Add images/screenshots
5. Publish on your blog

### For Email Marketing
1. Choose an email template
2. Customize the content
3. Replace placeholders (`[Name]`, `[your-url]`, etc.)
4. Use the HTML template for formatted emails
5. Send through your email marketing platform

## Customization Checklist

Before using any material, make sure to:
- [ ] Replace `[your-url]` with your actual website URL
- [ ] Replace `[your-email]` with your contact email
- [ ] Replace `[Name]` with recipient names (for emails)
- [ ] Add your actual images/screenshots
- [ ] Customize colors/branding if needed
- [ ] Review and adjust content for your specific audience
- [ ] Test all links before publishing

## Platform-Specific Notes

### Facebook
- Use square or landscape images (1200x628px recommended)
- Keep ad copy concise
- Use clear call-to-action buttons

### Instagram
- Use square (1:1) or vertical (4:5) images
- Keep captions engaging
- Use 5-10 relevant hashtags
- Post during peak hours (9 AM - 5 PM)

### Threads
- Keep posts conversational
- Use 2-3 hashtags maximum
- Engage with comments
- Post 1-2 times per day

### Blogs
- Add relevant images
- Include meta descriptions for SEO
- Use proper headings (H1, H2, H3)
- Add internal/external links

### Email
- Test on multiple email clients
- Check mobile responsiveness
- Verify all links work
- Include unsubscribe option
